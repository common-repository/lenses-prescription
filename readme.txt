Displays product options as a table for contact lenses prescription.


== Description ==

This plugin requires [Simple Product Options for WooCommerce](https://wordpress.org/plugins/product-options-for-woocommerce/) plugin to display the prescription options.

This plugin combines multiple Drop-down product options and displays them as the lenses prescription table.

How To Use It:
1. Install the Product Options plugin.
2. Import prescription options from a .csv file.
3. Select which options should be displayed as table on the edit product page.
[More details](http://hottons.com/demo/wp/lp/README.html#example)


**Main Features:**

* Combines product options and displays them as a table.


**Main Limitations:**

* Requires another plugin to display the product options.

== Installation ==

Requires  WordPress 4.7,  WooCommerce 3.0  and  Simple Product Options for WooCommerce 1.0

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `Lenses Prescription` from Plugins page.

Contact me by email <pektsekye@gmail.com> if you have questions or need help.
